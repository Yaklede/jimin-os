# 한국 부동산 리서치 가이드

## 목적

한국 부동산 시장을 공식 출처 기반으로 정리합니다. 이 문서는 투자 추천서가 아니라 리서치 품질 기준입니다.

## 기본 출처

| 영역 | 우선 출처 | 확인할 것 |
|---|---|---|
| 실거래가 | 국토교통부 실거래가 공개시스템, 공공데이터포털 실거래가 API | 지역, 계약년월, 거래유형, 면적, 층, 신고 기준 |
| 전월세 | 국토교통부 전월세 실거래가 API | 보증금, 월세, 계약 기간, 갱신 여부 |
| 통계 지수 | 한국부동산원 R-ONE | 매매/전세 가격지수, 지역 단위, 공표 주기 |
| 거시 지표 | 한국은행 ECOS, KOSIS | 금리, 물가, 인구, 가구, 소득 |

## 리서치 순서

1. 질문을 좁힙니다: 지역, 주택유형, 면적대, 기간, 거래유형.
2. 기준일을 적습니다: 데이터 조회일과 데이터 기준월을 분리합니다.
3. 출처별 정의를 적습니다: 실거래, 호가, 지수, 표본조사 여부.
4. 같은 단위로 비교합니다: 전용면적, 행정구역, 월 단위, 거래유형을 맞춥니다.
5. 변화율과 절대값을 같이 봅니다.
6. 공급, 입주, 금리, 전세가율, 거래량을 함께 봅니다.
7. 반대 시나리오를 적습니다.
8. 투자 추천이 아니라 관찰 가능한 사실과 확인 필요 항목으로 마무리합니다.

## 금지 표현

- 지금 사라
- 무조건 오른다
- 확실한 투자처
- 손해 볼 수 없다
- 단기 급등 보장

## 필수 산출물

- 요약
- 질문과 범위
- 데이터 출처와 기준일
- 주요 지표 표
- 해석
- 반대 시나리오
- 데이터 한계
- 추가 확인 질문
- 비추천 고지
